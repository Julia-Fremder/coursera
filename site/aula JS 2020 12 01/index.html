Funções JS

Basicamente, temos as seguintes funções ou os comandos das linguagens:
Funções internas - já parte da sintaxe da linguagem.
Funções externas - criadas pelo desenvolvedor.

Valor:

Função simples: executa alguma coisa simples.
Exemplo:function funcao01(){ var x = document.getElementById('valor1').value; var resultado = x * 2; alert('O dobro é: ' + resultado); }

Função com parâmetros: executa a função de acordo com o parâmetro atribuído.
Exemplos: function funcao02(numero){ var x = parseFloat(numero); alert('O dobro de ' + numero + 'é' + resultado) } funcao02(57), qnd clica no botão funcao02, aparece no alert 'o dobro de 57 é 114'
function funcao03 (x, y){ var resultado = parseFloat(x) * parseFloat(y); alert('o resultado é: ' + resultado); } funcao03(4, 9); --> aparece no alert 'o resultado é 36'.

Funções que retornam dados: podem receber parâmetros ou não. return true or false, aplica uma condição para a função ser executada, por exemplo o link 'clique aqui'.
function funcao05(){ var v1= document.getElementById('valor1').value; if (v1=='123'){ return true; }else { return false; } }--> dentro da tag link: a href="file02.html" onclick="return funcao05()">Clique aqui-->
